\section*{Introduction}
This report briefly explains the portfolio website I created for my assessment. The website presents my CV, contact information, and a short introduction about myself. It is designed with a dark theme and blood-red accents to match my personal style.

\section*{Website Description}
The website is built using simple HTML and CSS. It includes the following sections:
\begin{itemize}
    \item About Me
    \item CV (embedded and downloadable)
    \item Contact Information
\end{itemize}

The website is hosted using GitHub Pages. All files, including my CV, are stored in my GitHub repository.
